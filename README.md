# bbrf-dashboard

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

```
npm run build
```

#### Build image.
```
docker build . -t bbrf-dashboard;
```

#### Run image
```
docker run -d -p 80:80 bbrf-dashboard
```
